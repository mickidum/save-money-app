# Rest Api Node and Mongo

## Description
This is an Restful API for Node.js and Mongo. This is in the MVC format

##### Routing         : Express
##### ODM Database    : Mongoose
##### Authentication  : Passport, JWT

## Installation

#### Donwload Code | Clone the Repo

```
git clone {repo_name}
```

#### Install Node Modules
```
npm install
```
